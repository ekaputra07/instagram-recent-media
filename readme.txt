== Description ==
Display your most recent Instagram photos in a widget. This plugin means to be simple, easy to use and works.

No third party website and Instagram connect needed, just your website and Instagram.

To avoid request to Instagram server on every page load, this will only pull data from Instagram once a day.

== Installation ==
1. Unzip and upload `/instagram-recent-media/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set Instagram User ID and Client ID at Instagram settings page (Settings -> Instagram)
